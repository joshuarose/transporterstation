## Transporter Station

Open Source Web Application to assist Animal Rescue organizations in
moving animals from place to place with a volunteer staff

Roles:
    1. Organization
    2. Volunteer

Models:
    1. User
        a. username
        b. role
        c. password hash
    2. Animal
        a. name
        b. weight
        c. breed
        d. crated
        e. Special needs
        f. Runs
        e. image
    3. Run
        a. name
        b. Legs
        c. Date
        d. status
